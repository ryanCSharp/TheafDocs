## Theaf Bot

Theaf is a Discord Bot. Its purpose is a ROBLOX Utility Bot, it has a few commands for now.

### Commands

Here's a list of the commands that the bot has:

```markdown
# Assets API
- x>steal {AssetId}

# Users API
- x>search {username}
- x>idsearch {userId}
```
